Installation
============

Automatic with PyPI
-------------------

::

   $ pip install sdcclient

Manual (development only)
-------------------------

This method requires `uv`_ installed.

.. _uv: https://docs.astral.sh/uv/

::

    $ git clone https://github.com/sysdiglabs/sysdig-sdk-python.git
    $ cd python-sdc-client
    $ uv build
